Changelog

Every release, every change — documented in one place.

This log covers all notable changes to Outpost CMS. The format follows Keep a Changelog.

1.9.2 — 2026-03-08

Added

  • JSON Schema for theme.json — php/docs/schemas/theme.schema.json provides IDE autocomplete and validation for theme configuration files
  • Skeleton theme — developer reference theme with heavily commented templates demonstrating every Outpost template tag (managed, ships with every install)
  • $schema reference added to all three bundled theme.json files (Personal, Starter, Skeleton) for IDE integration

1.9.1 — 2026-03-08

Added

  • Changelog and Roadmap pages in developer documentation

1.9.0 — 2026-03-08

Added

  • Theme Update System — auto-updater safely updates bundled managed themes while preserving user modifications via hash-based conflict detection
  • Managed theme flag ("managed": true in theme.json) for Personal and Starter themes
  • .outpost-manifest.json generation in packaging script for file-level conflict detection
  • Theme update results display in Settings → Updates (installed, updated, conflicts)
  • “Managed by Outpost” label on managed themes in the Themes page
  • Delete protection for managed themes (button hidden in admin UI)
  • New theme auto-install during updates
  • Pre-compilation tag validation — balanced tag checking with line numbers for {% if %}, {% for %}, {% single %}
  • Source line tracking — /* @line:N */ markers in compiled templates for source mapping
  • Enhanced error display — admin-only diagnostics with source context, line highlighting, and friendly error messages

Changed

  • Duplicated themes have managed flag stripped — copies are fully user-owned
  • Themes API now includes managed boolean in list response

1.8.2 — 2026-03-08

Fixed

  • Body font no longer overrides heading font — engine and preview now pin non-customized font vars to their defaults
  • Live preview now updates in real-time — fixed postMessage failing silently on Svelte 5 proxy objects
  • “System Default” font choice now properly outputs system-ui stack

1.8.1 — 2026-03-08

Fixed

  • Customizer sidebar icon replaced with paintbrush
  • Calendar page header icon now uses standard wrapper
  • structuredClone crash on customizer page — Svelte 5 proxies replaced with JSON.parse(JSON.stringify())

Security

  • CSS injection hardening — type-specific validation for color (hex) and font (alphanumeric) values
  • postMessage origin check in customizer preview iframe
  • Sanitized theme slug in customizer export filename
  • Preview script validates CSS variable names and values before applying

1.8.0 — 2026-03-08

Added

  • Theme Customizer — visual editor for theme colors, fonts, logo, and favicon without touching code
  • Color palette editor with native color pickers and hex input
  • Font selector with curated Google Fonts library and live preview
  • Logo and favicon upload via Site Identity section
  • Live preview iframe with real-time CSS variable updates
  • Export/import customization presets as JSON
  • One-click reset to theme defaults

1.7.0 — 2026-03-07

Added

  • Multi-folder assignment — files can belong to multiple folders via junction table
  • Gallery from folders — {% for img in media_folder.slug %} template tag
  • Resizable detail sidebar with drag handle
  • Bulk folder creation (comma-separated names, max 50)
  • Right-click file context menu with folder operations
  • Role-based folder restrictions for editors
  • Folder slugs for template tag resolution

1.6.2 — 2026-03-07

Added

  • Bulk media operations — multi-select, shift-click range select, bulk delete, and bulk move-to-folder

1.6.1 — 2026-03-07

Fixed

  • Upload queue fired multiple times per file due to Svelte $effect reactive loop

1.6.0 — 2026-03-07

Added

  • WebP auto-conversion — JPEG and PNG uploads automatically converted to WebP
  • Focal point picker — click image preview to set focal point; {{ field | focal }} template filter
  • Media folder organization — folder sidebar, create/rename/delete, drag-to-folder, 3 levels max
  • Bulk upload with progress — upload queue with per-file progress bars, 2 concurrent, cancel support
  • MediaPicker folder browsing

1.5.2 — 2026-03-07

Fixed

  • Update checker now shows combined release notes for all versions between installed and latest

1.5.1 — 2026-03-07

Fixed

  • Update notification dot rendering fix — replaced with positioned green dot
  • Docs sidebar refactored to single shared sidebar.html

1.5.0 — 2026-03-06

Added

  • Review & Approval workflow — per-collection require_review toggle; editors submit for review, admins approve or reject
  • Editorial calendar — month-grid view of scheduled and published items
  • Bulk scheduling — select multiple items, apply publish date to all
  • Webhook events for review and scheduling lifecycle

1.4.0 — 2026-03-06

Added

  • User content directory — all user-owned data consolidated under outpost/content/
  • Auto-migration for existing sites with symlinks for URL compatibility

Security

  • Backup restore validates zip entry paths (zip slip prevention)
  • outpost_rmdir_recursive() no longer follows symlinks

1.3.2 — 2026-03-06

Fixed

  • Backups now include themes directory alongside database and uploads

1.3.1 — 2026-03-06

Added

  • Auto update check on every admin page load via auth/me
  • Sidebar update badge (green dot on Settings)

1.3.0 — 2026-03-06

Added

  • Backup & Restore — one-click backup of SQLite database + uploaded media
  • Restore from backup (super_admin only)
  • Automatic backups with configurable retention
  • Backup history with download and delete

1.2.0 — 2026-03-06

Added

  • Content Editor cleanup — restricted sidebar for editor role
  • Collection-scoped editors — restrict editors to specific collections

1.1.0 — 2026-03-06
