Meta Tags

Two dedicated helpers — data-outpost="title" and data-outpost="description" — output SEO-ready values with automatic fallback chains for both regular pages and collection single items.

<outpost-meta title="Default" />

<outpost-meta> — Title and Description

The <outpost-meta> self-closing element replaces both data-outpost="title" and data-outpost="description". It outputs the <title> tag and <meta name="description"> tag in one declaration. Default values are provided as attributes.

<outpost-meta title="My Blog" description="A blog about web development." />

This outputs:

<title>My Blog</title>
<meta name="description" content="A blog about web development.">

The same fallback chains from apply — if the current page or single item has a custom meta title or description set in the admin, those values take priority over the defaults provided in the attributes.

<outpost-seo> — Full SEO Block

The <outpost-seo /> element replaces the <outpost-seo /> tag. It outputs a complete SEO block including the <title>, meta description, Open Graph tags, and Twitter Card tags — all using the same fallback chains.

<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1">

 <outpost-seo />

 <link rel="stylesheet" href="/assets/style.css">
</head>

This single element outputs <title>, <meta name="description">, <meta property="og:title">, <meta property="og:description">, <meta property="og:type">, <meta name="twitter:card">, <meta name="twitter:title">, and <meta name="twitter:description"> — all populated from the same fallback chain described in above.

head.html Example

<!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1">

 <outpost-meta title="My Site" description="Welcome to my site." />

 <link rel="stylesheet" href="/assets/style.css">
</head>
<body>

Or, for the full SEO block including Open Graph and Twitter Card tags:

<!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1">

 <outpost-seo />

 <link rel="stylesheet" href="/assets/style.css">
</head>
<body>
