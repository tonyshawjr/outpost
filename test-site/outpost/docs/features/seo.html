SEO

Outpost handles SEO automatically. The <outpost-seo /> tag in your templates outputs title, description, Open Graph, Twitter Card, canonical URL, and JSON-LD — all pulled from the values you set in the admin. You just fill in the fields.

Where to update SEO in the admin

There are three places to manage SEO, from site-wide defaults down to individual pages and posts.

1. Site-wide defaults — Admin → Globals

These apply to every page on your site that doesn't have its own custom values set. Set them once and you're done.

Field nameWhat it doesExample
site_name Appended to every page title: Page Title — Site Name My Blog
site_tagline Fallback meta description for pages that have no description set Writing about design and code.
og_image Default social share image (1200×630px). Used when a page or post has no featured image. Upload via the image picker
twitter_handle Your Twitter/X handle for Twitter Card attribution. Include the @. @myhandle
