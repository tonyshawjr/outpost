Pagination

Split long lists across multiple pages. One option on the loop, one tag after it — pagination handles the rest automatically.

Setup

Add paginate="N" to a collection loop. The pagination controls are automatically rendered after the loop. That is all that is required.

<outpost-each collection="post" paginate="10" sort="published_at">
 <article>
 <h2 data-outpost="title">Post Title</h2>
 <p data-outpost="excerpt"></p>
 </article>
</outpost-each>

When paginate is set, the engine automatically appends <outpost-pagination /> after the loop. You can also place <outpost-pagination /> manually if you need it in a different position.
