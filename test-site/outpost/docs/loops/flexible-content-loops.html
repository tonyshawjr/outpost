Flexible Content Loops

Build dynamic page layouts from modular content blocks. Flexible content fields let editors compose pages from named layouts — each with its own set of sub-fields — enabling freeform page building within a structured schema.

What Is a Flexible Content Field?

Syntax

Use <outpost-each flexible="field_name"> to loop over a flexible content field. Inside the loop, access sub-fields with data-outpost attributes and read the layout name with data-outpost="layout".

<outpost-each flexible="field_name">
 <div data-outpost="layout"></div>
 <div data-outpost="sub_field"></div>
</outpost-each>

Rendering by Layout Type

Use data-layout attributes on child elements to conditionally render content based on the current block's layout type.

<outpost-each flexible="page_sections">

 <section class="hero" data-layout="hero">
 <h1 data-outpost="title"></h1>
 <p data-outpost="subtitle"></p>
 <img data-outpost="image" data-type="image" src="" alt="">
 </section>

 <blockquote class="testimonial" data-layout="testimonial">
 <div data-outpost="quote" data-type="richtext"></div>
 <cite data-outpost="author"></cite>
 </blockquote>

 <section class="prose" data-layout="text_block">
 <div data-outpost="body" data-type="richtext"></div>
 </section>

</outpost-each>

Example: Landing Page Builder

<outpost-each flexible="sections">

 <section class="hero" data-layout="hero">
 <div class="hero-inner">
 <h1 data-outpost="heading"></h1>
 <p class="hero-sub" data-outpost="subheading"></p>
 <a data-outpost="cta_url" data-type="link" href="#" class="btn btn-primary">
 <span data-outpost="cta_label">Get Started</span>
 </a>
 </div>
 <img data-outpost="background_image" data-type="image" src="" alt="" class="hero-bg">
 </section>

 <section class="testimonials" data-layout="testimonials">
 <blockquote>
 <p data-outpost="quote"></p>
 <footer>
 <img data-outpost="avatar" data-type="image" src="" alt="" class="avatar">
 <cite data-outpost="author"></cite>
 </footer>
 </blockquote>
 </section>

 <section class="cta-banner" data-layout="cta">
 <h2 data-outpost="heading"></h2>
 <p data-outpost="body_text"></p>
 <a data-outpost="button_url" data-type="link" href="#" class="btn btn-primary">
 <span data-outpost="button_text">Click Here</span>
 </a>
 </section>

</outpost-each>
