Channel Loops

Iterate over cached items from external data sources. Channels bring external data into your templates with the same for-loop syntax you already use for collections — regardless of whether the source is a REST API, RSS feed, or CSV file.

Basic Syntax

Syntax

Use <outpost-each channel="slug"> to loop over all cached items in a channel. Inside the loop, access item fields with data-outpost attributes.

<outpost-each channel="listings">
 <div class="listing">
 <h2 data-outpost="title"></h2>
 <p data-outpost="description"></p>
 <p class="price" data-outpost="price"></p>
 <img data-outpost-src="image_url" alt="">
 </div>
</outpost-each>

Real Estate Listing Page 

<main>
 <h1>Available Properties</h1>
 <div class="property-grid">
 <outpost-each channel="properties">
 <a class="property-card">
 <img data-outpost-src="image" alt="">
 <h2 data-outpost="title"></h2>
 <p class="price" data-outpost="price"></p>
 <p class="location">
 <span data-outpost="city"></span>,
 <span data-outpost="state"></span>
 </p>
 </a>
 </outpost-each>
 </div>
</main>

RSS Feed 

<outpost-each channel="blog_feed">
 <article class="feed-item">
 <h2><a data-outpost-href="link" data-outpost="title"></a></h2>
 <time data-outpost="pubDate"></time>
 <span class="author" data-outpost="author"></span>
 <p data-outpost="description"></p>
 </article>
</outpost-each>

Mixing Channels and Collections 

<section class="our-posts">
 <h2>Latest Articles</h2>
 <outpost-each channel="blog_feed">
 <a data-outpost-href="link" data-outpost="title"></a>
 </outpost-each>
</section>

<section class="external-listings">
 <h2>Featured Listings</h2>
 <outpost-each channel="mls">
 <div class="listing-card">
 <h3 data-outpost="address"></h3>
 <p data-outpost="price"></p>
 </div>
 </outpost-each>
</section>
