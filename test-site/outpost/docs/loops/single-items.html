Single Items

Fetch and render one specific collection item by its URL slug. This is the template tag that powers every individual post, project, product, or detail page.

Basic Syntax

How It Works

When a visitor hits a collection URL like /post/hello-world, the router extracts the slug (hello-world) and sets it as an internal parameter. The <outpost-single> tag reads that slug, queries the collection for a published item matching it, and renders the main block. If no published item with that slug exists, the <outpost-else> branch renders instead.
