Features

Everything Forge does to make theme building faster. Each feature maps to a real Outpost template concept.

Make Editable

The core action. Select any text, heading, image, or link and turn it into an admin-editable field. Forge auto-detects the field type — headings become text fields, images get the image filter, links get the link filter.

Your content stays on the page as a default value. Nothing disappears. The text you selected becomes the fallback that shows until the admin enters something new.

  • 10 field types: text, richtext, image, link, textarea, select, color, number, date, toggle
  • Page fields or global fields — one toggle
  • Auto-generated snake_case field names
  • Wrapping defaults preserve your content

Collection Loops

Got a grid of blog cards, team members, or portfolio items? Select the repeating block, right-click, and wrap it in a collection loop.

Pick your collection from a dropdown, set the item variable, add an optional limit or sort order, and Forge writes the entire for/endfor block for you.

  • Dropdown of all your collections
  • Custom item variable names
  • Optional limit and order-by fields
  • Preserves your HTML inside the loop

Conditionals

Wrap any section in an if/endif block to show or hide it based on a field value. Perfect for optional CTAs, seasonal banners, or member-only content.

Forge supports three operators: truthy (field has a value), equals, and not-equals. Complex logic without writing a single bracket by hand.

  • Truthy, equals, and not-equals operators
  • Works with page fields, globals, and item fields
  • Clean block syntax around your selection

Extract Partial

The most magical action in Forge. Select an entire navigation, header, or footer section. Choose "Extract Partial" and name it. Forge does the rest:

  • Creates the partial file in partials/
  • Moves your selected HTML into it
  • Replaces the selection with an include tag
  • Refreshes the file tree and opens the new file

One click turns inline HTML into a reusable component. Every page that includes it gets updates automatically.

Meta Tags & Forms

Two more actions for the things every page needs.

🏷️

Meta Tags

Select your page title or meta description and wrap it in SEO-friendly template tags. Forge detects whether you're inside a title element or a meta tag and sets the type automatically.

📋

Forms

Insert a form tag that renders any form you've built in the Outpost admin. Pick from a dropdown of existing forms or type a slug. The entire form is rendered from one template tag.

Under the Hood

How smart detection works

Forge reads the structure of your HTML and applies pattern matching to predict what you're trying to do.

<img>

Images

Any img, picture, or figure tag — or any element with a src attribute pointing to an image file — automatically suggests "Make Editable" with the image type pre-selected.

<a>

Links

Anchor tags with href attributes suggest "Make Editable" with the link filter, so the URL becomes editable in the admin.

<nav>

Structural elements

Nav, header, and footer elements suggest "Extract Partial" because they're almost always shared across pages and belong in their own file.

×3

Repeated siblings

If your selection contains multiple sibling elements with the same tag name, Forge suggests "Collection Loop" — because repeated markup usually means dynamic data.
