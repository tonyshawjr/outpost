How It Works

Three steps. No learning curve.

If you can right-click, you can use Forge. It meets you where you already work — inside the code editor.

1

Select your HTML

Highlight any element in the code editor — a heading, an image tag, a navigation block, a full section. Forge works with any selection size.

2

Right-click or press Cmd+E

The Forge menu appears with smart suggestions. It analyzes your selection and puts the most likely action at the top — images get "image", headings get "text".

3

Name and confirm

Give the field a name, choose any options, and click confirm. Your HTML is instantly wrapped in the correct template tag with the right filter.

Smart detection does the thinking

Forge doesn't just list every option — it reads your HTML and brings the right one to the top. Select an image tag and "image" is pre-selected. Select a nav element and "Extract Partial" jumps to first.

  • Images, pictures, and figures detected automatically
  • Links and anchor tags get the link filter
  • Headings default to text fields
  • Nav, header, footer suggest partial extraction
  • Repeated elements suggest collection loops

Extract partials instantly

Select your entire navigation, right-click, choose "Extract Partial", and name it "nav". Forge creates the file, moves the HTML into it, and replaces your selection with an include tag — all in one click.

The file tree refreshes, the new partial opens in a tab, and you're ready to keep building. No copy-paste, no switching files, no manual cleanup.
