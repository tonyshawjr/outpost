Output Tags — data-outpost

Output tags render field values into HTML. Add data-outpost="field_name" to any HTML element. The data-type attribute specifies the field type (defaults to text if omitted). The element's inner content serves as the default value. All output is HTML-escaped by default unless a type explicitly permits raw output.

Tag Description Example
data-outpost="field" Escaped text output. Registers the field in the admin editor. <h1 data-outpost="page_title"></h1>
Text with default Element content is the default value — shown as placeholder in admin and output when empty. <h1 data-outpost="hero_heading">Welcome</h1>
data-type="richtext" Raw HTML output — no escaping. For TipTap richtext editor content. <div data-outpost="body" data-type="richtext"></div>
data-type="image" Image URL. Opens the media picker in admin. <img data-outpost="hero_image" data-type="image" src="default.jpg">
data-type="link" URL input field in admin. <a data-outpost="cta_url" data-type="link" href="#">...</a>
data-type="textarea" Multi-line text — newlines converted to <br> tags. <p data-outpost="bio" data-type="textarea"></p>
data-type="select" Dropdown — editor chooses from predefined options in collection schema. <span data-outpost="category" data-type="select"></span>
data-type="color" Color picker — outputs a hex string like #14b8a6. <span data-outpost="brand_color" data-type="color">#000</span>
data-type="number" Numeric input — outputs the number as a string. <span data-outpost="max_items" data-type="number">0</span>
data-type="date" Date input — outputs ISO 8601 format: 2025-03-04. <time data-outpost="publish_date" data-type="date">2026-01-01</time>
data-type="or_body" Text/excerpt with auto-fallback from the body field (40 words) if empty. <p data-outpost="excerpt" data-type="or_body"></p>
data-scope="global" Site-wide global field value (escaped text). <span data-outpost="site_name" data-scope="global">My Site</span>
Global image Global image field — media picker. <img data-outpost="logo" data-scope="global" data-type="image" src="default.png">
<outpost-meta /> Page <title> and <meta name="description"> with fallback chains. <outpost-meta title="My Blog" description="A blog." />
Loop item field Loop variable field — escaped text. Used inside <outpost-each> or <outpost-single> blocks. <h2 data-outpost="title">Post Title</h2>
Loop item richtext Loop variable richtext — raw HTML output. <div data-outpost="body" data-type="richtext"></div>
Loop item URL Auto-built URL from the collection's url_pattern. <a data-outpost="url" data-type="link">Read more</a>
Loop item date Formatted publish date (e.g. "March 4, 2026"). <time data-outpost="published_at"></time>

Block Tags — <outpost-*>

Block tags control template logic — loops, conditionals, includes, and SEO output.

Collection Loops

Tag Description Example
<outpost-each collection="slug"> Iterate over all published items in a collection. <outpost-each collection="post">
limit="N" Limit to N most recent items. <outpost-each collection="post" limit="5">
sort="field" Order by a specific field, descending. <outpost-each collection="post" sort="published_at">
paginate="N" Paginate — show N per page. Add <outpost-pagination /> after the loop. <outpost-each collection="post" paginate="10">
filteredby="param" Filter items by a URL query parameter value. <outpost-each collection="post" filteredby="category">
related="var" Show items related to the current single item. <outpost-each collection="post" related="post" limit="3">
<outpost-each ... empty> Empty state — rendered when a loop has no items. <outpost-each collection="post" empty><p>No posts yet.</p></outpost-each>

Single Items

Tag Description Example
<outpost-single collection="slug"> Fetch a single item by the URL slug parameter. <outpost-single collection="post">
<outpost-single ... else> Rendered when the item is not found (404 fallback). <outpost-single collection="post" else><p>Not found.</p></outpost-single>

Folder, Menu, Gallery, Repeater, Flexible Content, and Relationship Loops

Tag Description Example
<outpost-each folder="slug"> Iterate over labels in a folder. <outpost-each folder="tags">
<outpost-menu name="slug"> Iterate over items in a navigation menu. <outpost-menu name="main">
<outpost-each gallery="field"> Iterate over images in a gallery field. <outpost-each gallery="photos">
<outpost-each repeat="field"> Iterate over rows in a repeater field. <outpost-each repeat="features">
<outpost-each flexible="field"> Iterate over blocks in a flexible content field. Use data-layout to check the layout type. <outpost-each flexible="page_sections">
<outpost-each relationship="field"> Iterate over items in a relationship field. Returns full item data from the target collection. <outpost-each relationship="related_posts">
<outpost-pagination /> Renders pagination navigation after a paginate loop. <outpost-pagination />
<outpost-seo /> Outputs <title>, meta description, Open Graph, Twitter Card, canonical URL, and JSON-LD. Replaces manual meta tags. <outpost-seo />
<outpost-form slug="slug" /> Renders a builder-defined form with all fields, labels, validation, honeypot, and submit button. <outpost-form slug="contact" />

Conditionals

Tag Description Example
<outpost-if field="name" exists> True when a page field has a non-empty value. <outpost-if field="hero_image" exists>...</outpost-if>
scope="global" True when a global field has a value. <outpost-if field="twitter_url" scope="global" exists>...</outpost-if>
equals="value" Strict string equality check. <outpost-if field="category" equals="featured">
not-equals="value" String inequality check. <outpost-if field="status" not-equals="archived">
<outpost-if admin> True only when a logged-in admin is previewing the page. <outpost-if admin><div class="hint">...</div></outpost-if>
<outpost-else> Else branch inside any <outpost-if> block. <outpost-else><p>No image.</p></outpost-else>

Includes

Tag Description Example
<outpost-include partial="name" /> Insert a partial from partials/name.html. <outpost-include partial="nav" />

Example: Blog Post Card

<outpost-each collection="post" limit="6">
  <article class="card">
    <outpost-if field="featured_image" exists>
      <a href="" data-outpost="url">
        <img data-outpost="featured_image" data-type="image" src="" alt="">
      </a>
    </outpost-if>
    <div class="card-body">
      <h2><a href="" data-outpost="url">
        <span data-outpost="title">Post title</span>
      </a></h2>
      <p data-outpost="excerpt" data-type="or_body"></p>
      <time data-outpost="published_at"></time>
    </div>
  </article>
</outpost-each>

Example: Hero Section with Globals

<section class="hero">
  <h1 data-outpost="hero_heading">We build things people love.</h1>
  <div data-outpost="hero_body" data-type="richtext"></div>
  <outpost-if field="cta_url" exists>
    <a data-outpost="cta_url" data-type="link" href="#" class="btn">
      <span data-outpost="cta_label">Get started</span>
    </a>
  </outpost-if>
</section>

<footer>
  <span data-outpost="site_name" data-scope="global">My Site</span>
  <outpost-if field="twitter_url" scope="global" exists>
    <a data-outpost="twitter_url" data-scope="global" data-type="link" href="#">Twitter</a>
  </outpost-if>
</footer>
