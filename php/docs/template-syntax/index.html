Output Tags — {{ }}

Output tags render field values into HTML. All output is HTML-escaped by default unless a filter explicitly permits raw output.

Tag Description Example
{{ field }} Escaped text output. Registers the field in the admin editor. <h1>{{ page_title }}</h1>
{{ field }}Default{{ /field }} With default value — shown as placeholder in admin and output when empty. Wrapping syntax is recommended; inline {{ field "Default" }} also works. {{ hero_heading }}Welcome{{ /hero_heading }}
{{ field | raw }} Raw HTML output — no escaping. For TipTap richtext editor content. <div>{{ body | raw }}</div>
{{ field | image }} Image URL. Opens the media picker in admin. <img src="{{ hero_image | image }}">
{{ field | link }} URL input field in admin. <a href="{{ cta_url | link }}">...</a>
{{ field | textarea }} Multi-line text — newlines converted to <br> tags. <p>{{ bio | textarea }}</p>
{{ field | select }} Dropdown — editor chooses from predefined options in collection schema. <span>{{ category | select }}</span>
{{ field | toggle }} Boolean field — use in {% if %}, not as direct output. {% if featured %}...{% endif %}
{{ field | color }} Color picker — outputs a hex string like #14b8a6. style="color: {{ brand_color | color }};"
{{ field | number }} Numeric input — outputs the number as a string. {{ max_items | number }}
{{ field | date }} Date input — outputs ISO 8601 format: 2025-03-04. {{ publish_date | date }}
{{ field | focal }} Focal point as CSS X% Y%. For object-position / background-position. style="object-position: {{ hero | focal }};"
{{ field | or_body }} Text/excerpt with auto-fallback from the body field (40 words) if empty. <p>{{ excerpt | or_body }}</p>
{{ @global }} Site-wide global field value (escaped text). {{ @site_name }}
{{ @global | raw }} Global field as raw HTML. {{ @footer_text | raw }}
{{ @global | image }} Global image field — media picker. <img src="{{ @logo | image }}">
{{ meta.title }}Default{{ /meta.title }} Page <title> value with fallback chain. Outputs raw text, not the tag. <title>{{ meta.title }}My Blog{{ /meta.title }}</title>
{{ meta.description }}Default{{ /meta.description }} <meta name="description"> content with fallback chain. <meta name="description" content="{{ meta.description }}{{ /meta.description }}">
{{ item.field }} Loop variable field — escaped text. Used inside {% for %} blocks. {{ post.title }}
{{ item.field | raw }} Loop variable richtext — raw HTML output. {{ post.body | raw }}
{{ item.url }} Auto-built URL from the collection's url_pattern. <a href="{{ post.url }}">Read more</a>
{{ item.slug }} The item's URL slug. {{ post.slug }}
{{ item.published_at }} Formatted publish date (e.g. "March 4, 2026"). {{ post.published_at }}
{{ item.id }} Database integer ID. {{ post.id }}
{{ item.status }} Item status: published, draft, or scheduled. {% if post.status == "published" %}
{{ item.created_at }} Row creation timestamp. {{ post.created_at }}
{{ item.updated_at }} Last-modified timestamp. {{ post.updated_at }}

Block Tags — {% %}

Block tags control template logic — loops, conditionals, includes, and comments.

Collection Loops

Tag Description Example
{% for x in collection.slug %} Iterate over all published items in a collection. {% for post in collection.post %}
{% for x in collection.slug limit:5 %} Limit to N most recent items. {% for post in collection.post limit:5 %}
{% for x in collection.slug orderby:field %} Order by a specific field, descending. {% for p in collection.post orderby:title %}
{% for x in collection.slug paginate:10 %} Paginate — show 10 per page. Add {% pagination %} after the loop. {% for post in collection.post paginate:10 %}
{% for x in collection.slug filteredby:param %} Filter items by a URL query parameter value. {% for post in collection.post filteredby:category %}
{% for x in collection.slug related:var %} Show items related to the current single item. {% for post in collection.post related:current %}
{% else %} Empty state — rendered when a loop has no items. {% else %}<p>No posts yet.</p>
{% endfor %} Close a {% for %} block. {% endfor %}

Single Items

Tag Description Example
{% single x from collection.slug %} Fetch a single item by the ?slug= query parameter. {% single post from collection.post %}
{% else %} Rendered when the item is not found (404 fallback). {% else %}<p>Post not found.</p>
{% endsingle %} Close a {% single %} block. {% endsingle %}

Folder, Menu, Gallery, Repeater, Flexible Content, and Relationship Loops

Tag Description Example
{% for x in folder.slug %} Iterate over labels in a folder. {% for tag in folder.tags %}
{% for x in menu.slug %} Iterate over items in a navigation menu. {% for item in menu.main %}
{% for x in gallery.field %} Iterate over images in a gallery field. {% for img in gallery.photos %}
{% for x in repeater.field %} Iterate over rows in a repeater field. {% for row in repeater.features %}
{% for x in flexible.field %} Iterate over blocks in a flexible content field. Use block.layout to check the layout type. {% for block in flexible.page_sections %}
{% for x in relationship.field %} Iterate over items in a relationship field. Returns full item data from the target collection. {% for post in relationship.related_posts %}
{% pagination %} Renders pagination navigation after a paginate: loop. {% pagination %}
{% seo %} Outputs <title>, meta description, Open Graph, Twitter Card, canonical URL, and JSON-LD. Replaces manual meta tags. {% seo %}
{% form 'slug' %} Renders a builder-defined form with all fields, labels, validation, honeypot, and submit button. {% form 'contact' %}

Conditionals

Tag Description Example
{% if field %} True when a page field has a non-empty value. {% if hero_image %}...{% endif %}
{% if @global %} True when a global field has a value. {% if @twitter_url %}...{% endif %}
{% if item.field %} True when a loop variable field is non-empty. {% if post.featured_image %}...{% endif %}
{% if item.field == "value" %} Strict string equality check. {% if post.category == "featured" %}
{% if item.field != "value" %} String inequality check. {% if post.status != "archived" %}
{% if admin %} True only when a logged-in admin is previewing the page. {% if admin %}<div class="hint">...</div>{% endif %}
{% else %} Else branch inside any {% if %} block. {% else %}<p>No image.</p>
{% endif %} Close an {% if %} block. {% endif %}

Includes & Comments

Tag Description Example
{% include 'partial-name' %} Insert a partial from partials/partial-name.html. {% include 'nav' %}
{# comment #} Template comment — stripped entirely at compile time, never reaches the browser. {# TODO: add sidebar #}
