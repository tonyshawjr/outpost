Folder Loops

Iterate over the labels in a folder — categories, tags, topics, or any classification system you define. Use folder loops to build filter navigation, tag clouds, and category index pages.

Basic Syntax

The folder slug (e.g., categories, tags) is set when you create the folder in Admin → Folders. Use it in the folder attribute of <outpost-each>.

<outpost-each folder="categories">
 <a data-outpost="slug" data-type="link">
 <span data-outpost="name">Category Name</span>
 (<span data-outpost="count">0</span>)
 </a>
</outpost-each>

Each folder item provides three fields: name (display name), slug (URL-safe identifier), and count (number of published items with this label).

Combining Folder Nav with a Filtered Loop 

<div class="blog-layout">

 <aside class="sidebar">
 <h3>Categories</h3>
 <outpost-each folder="categories">
 <a data-outpost="slug" data-type="link">
 <span data-outpost="name">Category</span>
 (<span data-outpost="count">0</span>)
 </a>
 </outpost-each>
 </aside>

 <main class="main-col">
 <outpost-each collection="post" paginate="10" sort="published_at" filter="category">
 <article>
 <h2><a data-outpost="url" data-type="link"><span data-outpost="title">Title</span></a></h2>
 <p data-outpost="excerpt" data-type="or_body">Excerpt</p>
 </article>
 </outpost-each>

 <outpost-each collection="post" empty>
 <p>No posts in this category.</p>
 </outpost-each>

 <outpost-pagination />
 </main>

</div>

Tag Cloud 

<div class="tag-cloud">
 <outpost-each folder="tags">
 <a data-outpost="slug" data-type="link" class="tag" data-bind="data-count:count">
 <span data-outpost="name">Tag</span>
 </a>
 </outpost-each>
</div>
