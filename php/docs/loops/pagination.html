Pagination

Split long lists across multiple pages. One option on the loop, one tag after it — pagination handles the rest automatically.

Setup

Add paginate:N to a collection loop and place {% pagination %} immediately after the closing {% endfor %}. That is all that is required.

{% for post in collection.post paginate:10 %}
  <article>
    <h2><a href="{{ post.url }}">{{ post.title }}</a></h2>
    <p>{{ post.excerpt | or_body }}</p>
  </article>
{% endfor %}

{% pagination %}
Note — {% pagination %} must appear after the loop that uses paginate:N. It reads internal state set by the loop and renders nothing if placed before it or on a page with no paginated loop.

URL Format

The first page has no query parameter — it is the default URL for the page. Subsequent pages append ?page=N:

PageURL
1 (first)/blog
2/blog?page=2
3/blog?page=3

What the Pagination Tag Renders

{% pagination %} outputs a <nav> element with numbered page links, Previous and Next buttons, and ellipsis spacers for large page counts. It renders nothing at all when there is only one page.

<nav class="pagination">
  <a href="/blog" class="page-link">Previous</a>
  <a href="/blog" class="page-link page-current">1</a>
  <a href="/blog?page=2" class="page-link">2</a>
  <a href="/blog?page=3" class="page-link">3</a>
  <span class="page-ellipsis">...</span>
  <a href="/blog?page=9" class="page-link">9</a>
  <a href="/blog?page=2" class="page-link">Next</a>
</nav>

Styling Pagination

Add these rules to your theme's style.css to get clean, functional pagination. Adjust colors to match your design.

.pagination {
  display: flex;
  align-items: center;
  gap: 4px;
  margin: 40px 0;
  flex-wrap: wrap;
}

.page-link {
  padding: 8px 14px;
  border: 1px solid #e4e4e7;
  border-radius: 6px;
  text-decoration: none;
  color: #52525b;
  font-size: 14px;
  transition: border-color 0.15s, color 0.15s;
}

.page-link:hover {
  border-color: #a1a1aa;
  color: #18181b;
}

.page-link.page-current {
  background: #14b8a6;
  color: #fff;

  font-weight: 600;
}

.page-ellipsis {
  padding: 8px 6px;
  color: #a1a1aa;
  font-size: 14px;
}

Query String Preservation

When a folder filter is active, pagination links automatically carry the filter forward. A visitor browsing /blog?category=technology&page=3 gets pagination links that keep category=technology in every URL.

?category=technology          ← page 1
?category=technology&page=2  ← page 2
?category=technology&page=3  ← page 3

This works automatically — no extra configuration needed. Any query parameters present on the current request are preserved in pagination links, except for page itself which is managed by the tag.

paginate:N vs limit:N

These two options serve different purposes and cannot be used together.

OptionUse whenPagination navReads ?page=
paginate:N Building a browsable archive page Yes — via {% pagination %} Yes
limit:N Sidebar widgets, homepage previews, "top 5" lists No No
Warning — Do not use both paginate:N and limit:N on the same loop. The behavior is undefined. Use one or the other.

Combining with Filters and Sort

Pagination works alongside all other loop options. The total count respects the active filter, so pagination reflects only the pages within that subset.

{% for post in collection.post paginate:10 orderby:published_at filteredby:category %}
  <article>
    <h2><a href="{{ post.url }}">{{ post.title }}</a></h2>
  </article>
{% else %}
  <p>No posts in this category.</p>
{% endfor %}

{% pagination %}

Edge Cases

Fewer Items than the Page Size

If the collection has 7 published items and paginate:10 is set, all 7 items show on page 1 and {% pagination %} renders nothing — there is no page 2 to navigate to.

Out-of-Range Page Numbers

Requesting ?page=999 when only 3 pages exist returns an empty loop (the {% else %} branch fires). Pagination still renders, showing the correct page count.

Placing {% pagination %} in a Partial

You can include the pagination tag inside a partial, but the partial must be included after the loop closes — not before or inside it.

{% for post in collection.post paginate:10 %}
  ...
{% endfor %}

{% include 'pagination-bar' %}  {# partials/pagination-bar.html contains {% pagination %} #}
