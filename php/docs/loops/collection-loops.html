Collection Loops

Iterate over published items in any collection. The most-used tag in every Outpost site — list posts, products, events, portfolio pieces, or any structured content you define.

Basic Syntax

Basic Collection Loop

Use the <outpost-each> element with a collection attribute. Inside the loop, bind fields to elements using data-outpost.

<outpost-each collection="post">
 <article>
 <h2 data-outpost="title">Post Title</h2>
 <a data-outpost="url" data-type="link">
 <img data-outpost="featured_image" data-type="image">
 </a>
 <p data-outpost="excerpt">Excerpt text</p>
 <time data-outpost="published_at" data-bind="datetime:published_at">March 4, 2026</time>
 <div data-outpost="body" data-type="richtext">Body content</div>
 </article>
</outpost-each>

The text content of each element (e.g. "Post Title") serves as the default value and is visible when previewing the HTML file in a browser without the CMS.

Field Binding

SyntaxNotes
data-outpost="title"<h2 data-outpost="title">Default</h2>Text content is replaced
data-outpost="body" data-type="richtext"<div data-outpost="body" data-type="richtext"></div>Rich HTML output
data-outpost="featured_image"<img data-outpost="featured_image" data-type="image">Sets the src attribute
data-outpost="url"<a data-outpost="url" data-type="link">...</a>Sets the href attribute
data-outpost="excerpt" data-type="or_body"<p data-outpost="excerpt" data-type="or_body"></p>Falls back to body text

Attribute Binding with data-bind

Use data-bind to set HTML attributes from field values, rather than replacing text content.

<time data-outpost="published_at" data-bind="datetime:published_at">March 4, 2026</time>
<img data-outpost="featured_image" data-type="image" data-bind="alt:title">

The format is data-bind="attribute:field". Multiple bindings can be comma-separated.

Empty Loop Fallback

Add a sibling <outpost-each> with the empty attribute. It renders when the collection has no published items.

<outpost-each collection="post">
 <article>
 <h2 data-outpost="title">Post Title</h2>
 </article>
</outpost-each>

<outpost-each collection="post" empty>
 <p>No posts published yet.</p>
</outpost-each>

Loop Options

Options are set as attributes on the <outpost-each> element.

<!-- Limit to 6 items -->
<outpost-each collection="post" limit="6">...</outpost-each>

<!-- Sort by field, descending -->
<outpost-each collection="post" sort="published_at" order="desc">...</outpost-each>

<!-- Paginate -->
<outpost-each collection="post" paginate="10" sort="published_at">...</outpost-each>
<outpost-pagination />

<!-- Filter by folder label -->
<outpost-each collection="post" filter="category">...</outpost-each>

<!-- Related items (inside an outpost-single) -->
<outpost-each collection="post" related limit="3">...</outpost-each>

<!-- Combine options freely -->
<outpost-each collection="post" paginate="10" sort="published_at" filter="category">
 ...
</outpost-each>

Full Blog List Example 

<main class="blog-main">
 <header class="blog-header">
 <h1>Blog</h1>
 </header>

 <div class="post-grid">
 <outpost-each collection="post" paginate="12" sort="published_at" filter="category">
 <article class="post-card">
 <a data-outpost="url" data-type="link" class="post-card-image">
 <img data-outpost="featured_image" data-type="image" data-bind="alt:title">
 </a>
 <div class="post-card-body">
 <h2><a data-outpost="url" data-type="link"><span data-outpost="title">Post Title</span></a></h2>
 <p data-outpost="excerpt" data-type="or_body">Excerpt text</p>
 <time data-outpost="published_at" data-bind="datetime:published_at">March 4, 2026</time>
 </div>
 </article>
 </outpost-each>

 <outpost-each collection="post" empty>
 <p class="empty-state">No posts found in this category.</p>
 </outpost-each>
 </div>

 <outpost-pagination />
</main>

Multiple Collections on One Page 

<section class="featured-posts">
 <h2>Latest Articles</h2>
 <outpost-each collection="post" limit="3" sort="published_at">
 <a data-outpost="url" data-type="link">
 <span data-outpost="title">Post Title</span>
 </a>
 </outpost-each>
</section>

<section class="upcoming-events">
 <h2>Upcoming Events</h2>
 <outpost-each collection="event" limit="4" sort="date">
 <div>
 <strong data-outpost="title">Event Name</strong>
 <span data-outpost="date">Date</span>
 <a data-outpost="url" data-type="link">Details</a>
 </div>
 </outpost-each>
</section>
